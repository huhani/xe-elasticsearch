인덱스 설정

재생성 도움말

documents, comments, document_extra_vars 인덱스를 재생성합니다. 이미 생성된 인덱스가 있을 경우 삭제 후 재생성합니다.

초 갱신 도움말

documents, comments, document_extra_vars 인덱스의 refresh_interval 값을 변경합니다. 기본값: 1초.
0초 : null, -1초 : no index refresh
